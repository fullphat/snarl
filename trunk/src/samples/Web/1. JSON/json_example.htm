Including support for Snarl via JSON is incredibly simple

Just reference the "snarl.js" file included in this example, then...

Create a notification class object:

myClass = new Snarl.NotificationType('notification_class_here', true);

Register with Snarl:

Snarl.register('your_website_name', [myClass]);

Display a notification:

Snarl.notify(myClass, 'a_title', 'some_text', Snarl.Priority.VeryLow, false);

Try it for yourself

(Remember that Snarl must be running on the local computer and Network->Listen for incoming JSON Notifications? must be ON)
